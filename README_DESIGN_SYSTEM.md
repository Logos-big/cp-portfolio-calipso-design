# Дизайн-система и макет из Figma

## ⚠️ ВАЖНОЕ ПРАВИЛО

**Всегда следуйте СТРОГО по макету и дизайн-системе из Figma.**

## Принципы работы

### 1. Только то, что есть в макете

- ✅ Генерируются ТОЛЬКО элементы, которые есть в макете Figma
- ✅ Применяются ТОЛЬКО стили из макета Figma
- ❌ НЕ добавляются элементы, которых нет в макете
- ❌ НЕ добавляются стили, не указанные в макете
- ❌ НЕ добавляются анимации или эффекты, не указанные в дизайне

### 2. Точное соответствие структуре

Скрипт автоматически:
- Воспроизводит точную иерархию элементов из Figma
- Сохраняет порядок элементов как в макете
- Использует те же типы элементов (FRAME, TEXT, RECTANGLE и т.д.)

### 3. Точное соответствие стилям

Из макета извлекаются и применяются:
- ✅ Точные размеры (width, height)
- ✅ Точные цвета (RGB → HEX)
- ✅ Точные шрифты (family, size, weight, line-height, letter-spacing)
- ✅ Точные отступы (padding, gap, margin)
- ✅ Точное выравнивание (align-items, justify-content)
- ✅ Точные layout свойства (flex-direction, display)

### 4. Дизайн-система

Используются только значения из дизайн-системы Figma:
- Цвета из палитры Figma
- Шрифты из типографики Figma
- Spacing из дизайн-системы Figma

## Как работает автоматическое обновление

1. **Получение данных из Figma**
   - Запрашивается ТОЛЬКО узел "Main" (node-id=67:5539)
   - Получаются все свойства элементов
   - Сохраняются в `figma-data.json`

2. **Генерация HTML**
   - Парсится структура узла "Main"
   - Рекурсивно генерируются элементы
   - Применяются стили строго из данных Figma

3. **Результат**
   - HTML точно соответствует макету
   - Все стили точно соответствуют дизайну
   - Ничего лишнего не добавляется

## Проверка соответствия

После обновления проверьте:
1. ✅ Структура HTML = структура в Figma
2. ✅ Все размеры точно совпадают
3. ✅ Все цвета точно совпадают
4. ✅ Все шрифты точно совпадают
5. ✅ Все отступы точно совпадают

## Обновление макета

Когда вы обновляете макет в Figma:

1. Обновите макет "Main" в Figma
2. Напишите: **"обновил Main"**
3. Скрипт автоматически:
   - Получит свежие данные
   - Обновит HTML строго по макету
   - Обновит сайт на GitHub Pages

## Файлы

- `scripts/auto-update.ps1` - основной скрипт (следует строго макету)
- `figma-data.json` - данные из Figma
- `index.html` - сгенерированный HTML (строго по макету)
- `styles/figma-variables.css` - CSS переменные из дизайн-системы
