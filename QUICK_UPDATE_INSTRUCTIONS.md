# Быстрое обновление

## Автоматическое обновление

Когда вы обновили макет в Figma, просто напишите **"обновил Main"** (или "обновил тото"), и я автоматически обновлю страницу.

## Ручное обновление

Если автоматическое обновление не сработало, запустите один из скриптов:

### Вариант 1: PowerShell скрипт
```powershell
.\scripts\auto-update.ps1
```

### Вариант 2: BAT файл (двойной клик)
Просто дважды кликните на файл `update.bat` в корне проекта.

### Вариант 3: Python скрипт (если установлен Python)
```powershell
python scripts\update_from_figma.py
```

## Что обновляется

После запуска скрипта обновляются:
- ✅ `figma-data.json` - свежие данные из Figma
- ✅ `styles/figma-variables.css` - CSS переменные (цвета, шрифты, отступы)
- ✅ `index.html` - HTML структура страницы
- ✅ `styles/main.css` - стили элементов

## Текущий статус

Страница обновлена на основе последних данных из `figma-data.json`.
