# СМ2 - Резервная копия сайта

Это резервная копия сайта, созданная для возможности возврата к стабильной версии.

## Дата создания
Создано: $(Get-Date -Format "yyyy-MM-dd HH:mm:ss")

## Содержимое
- `index.html` - главная страница сайта
- `admin.html` - админка с визуальным редактором
- `project.html` - страница проекта
- `styles/` - все стили CSS (main.css, project.css, admin-editor.css)
- `images/` - все изображения проекта
- `lottie/` - анимации Lottie
- `data/` - данные проектов (projects.json)
- `scripts/` - все JavaScript файлы (admin-editor.js, project.js)
- `projects/` - папки с проектами
- `start-server.bat` и `start-server.ps1` - скрипты для запуска локального сервера
- `README-SERVER.md` - инструкции по запуску сервера

## Функционал этой версии

### Админка (admin.html)
- Визуально повторяет главную страницу
- Режим редактирования с drag and drop
- Перетаскивание проектов для изменения порядка
- Создание новых проектов
- Автоматическое сохранение в localStorage
- Кнопка "Подтвердить изменения"

### Главная страница (index.html)
- Карусель проектов с автопрокруткой
- Загрузка данных из localStorage
- Сортировка проектов по полю `order`

### Страница проекта (project.html)
- Удаление проекта
- Загрузка фото в модалку
- Синхронные карусели сравнения (чертежи, фото, рендеры)

## Как использовать
1. Откройте папку СМ2
2. Запустите `start-server.bat` или `start-server.ps1`
3. Откройте в браузере `http://localhost:8000`

## Примечание
Эта версия сохранена как стабильная точка восстановления. Если что-то сломается в основной версии, вы можете продолжить работу с этой копией.
