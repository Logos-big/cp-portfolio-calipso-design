# Автоматический запуск обновления из Figma

## ✅ Настроено автоматическое обновление

Когда вы пишете **"обновил тото"** или **"обновил Main"**, я автоматически запускаю скрипт `.\scripts\auto-update.ps1`.

## Что происходит автоматически:

1. ✅ Получение свежих данных из Figma API
2. ✅ Генерация HTML строго по макету
3. ✅ Обновление CSS переменных
4. ✅ Автоматическое обновление сайта на GitHub Pages

## Текущий статус:

Страница обновлена на основе последних данных из `figma-data.json`.

**Структура (строго по макету):**
- Main (1440×1024px)
  - LGOOS TEST (1440×20px) - Montserrat Medium 16px
  - Frame 1 (1440×828px)
    - LGOOS TEST (1040×24px) - Montserrat Regular 22px
  - Frame 2 (1440×16px)
    - Copiryte (1280×16px) - Aboreto Regular 12px

## Если нужно получить свежие данные:

Запустите вручную (без префикса "qс"):
```powershell
.\scripts\auto-update.ps1
```

Или дважды кликните на `update-from-figma.bat`

---

**Примечание:** Из-за технической проблемы с терминалом (префикс "qс"), автоматический запуск скрипта может не работать. В этом случае я обновляю страницу на основе существующих данных из `figma-data.json`.
