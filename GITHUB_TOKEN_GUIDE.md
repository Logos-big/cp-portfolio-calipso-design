# Как получить Personal Access Token для GitHub

## Шаг 1: Войдите в GitHub
Зайдите на [github.com](https://github.com) и войдите в свой аккаунт

## Шаг 2: Откройте настройки
1. Нажмите на ваш аватар в правом верхнем углу
2. Выберите **Settings** (Настройки)

## Шаг 3: Перейдите в Developer settings
1. В левом меню прокрутите вниз
2. Нажмите **Developer settings** (в самом низу)

## Шаг 4: Создайте новый токен
1. В левом меню выберите **Personal access tokens**
2. Нажмите **Tokens (classic)** или **Fine-grained tokens**
3. Нажмите **Generate new token** → **Generate new token (classic)**

## Шаг 5: Настройте токен
1. **Note** (Название): введите описание, например "CP Portfolio Project"
2. **Expiration** (Срок действия): выберите срок (можно выбрать "No expiration" для постоянного токена)
3. **Select scopes** (Выберите права):
   - ✅ **repo** (полный контроль над репозиториями) - это самое важное!
   - ✅ **workflow** (если используете GitHub Actions)

## Шаг 6: Создайте токен
1. Прокрутите вниз и нажмите **Generate token**
2. **ВАЖНО!** Скопируйте токен сразу - он показывается только один раз!
3. Сохраните токен в безопасном месте

## Шаг 7: Используйте токен
При выполнении `git push`:
- **Username**: ваш GitHub username (Logos-big)
- **Password**: вставьте скопированный токен (НЕ ваш обычный пароль!)

---

## Альтернатива: GitHub CLI (более простой способ)

Если хотите упростить процесс, установите GitHub CLI:

1. Скачайте с [cli.github.com](https://cli.github.com)
2. Установите
3. Выполните:
```powershell
gh auth login
```
4. Следуйте инструкциям в терминале

После этого `git push` будет работать автоматически!
