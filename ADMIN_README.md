# Инструкция по использованию админки

## Как открыть админку

1. **Запустите локальный сервер** (обязательно!):
   ```bash
   # Вариант 1: Используйте готовый скрипт
   start-server.bat
   
   # Вариант 2: Через Python
   python -m http.server 8000
   ```

2. **Откройте в браузере**:
   ```
   http://localhost:8000/admin.html
   ```

   ИЛИ нажмите кнопку "Админка" в правом нижнем углу главной страницы.

## Важно!

⚠️ **Админка НЕ будет работать при открытии через `file://` протокол!**

Обязательно используйте локальный HTTP сервер (`http://localhost:8000`).

## Функции админки

### 1. Режим редактирования
- Нажмите кнопку **"Режим редактирования"** в правом верхнем углу
- Элементы карусели станут перетаскиваемыми
- Появится иконка "⋮⋮" на каждом элементе

### 2. Изменение порядка проектов
- В режиме редактирования перетащите элементы карусели
- Порядок автоматически сохраняется в памяти
- Появится индикатор "Есть несохраненные изменения"

### 3. Сохранение изменений
- Нажмите **"Подтвердить изменения"**
- Изменения сохранятся в `data/projects.json`
- Страница автоматически обновится

### 4. Отмена изменений
- Нажмите **"Отмена"** для отмены всех несохраненных изменений

### 5. Футер с папками проектов
- Внизу страницы отображаются все проекты
- Каждая папка показывает:
  - Название проекта
  - Количество изображений
  - Миниатюры (сетка 3x3)
- Клик по папке открывает страницу проекта

## Решение проблем

### Админка не открывается
1. Убедитесь, что запущен локальный сервер
2. Откройте консоль браузера (F12) и проверьте ошибки
3. Проверьте, что все файлы на месте:
   - `admin.html`
   - `scripts/admin-editor.js`
   - `styles/admin-editor.css`
   - `data/projects.json`

### Карусель не загружается
1. Проверьте консоль браузера на ошибки
2. Убедитесь, что `data/projects.json` содержит данные
3. Проверьте, что изображения доступны по указанным путям

### Изменения не сохраняются
- В текущей версии сохранение работает через `fetch PUT`
- Для реального сохранения в файл нужен сервер с API
- Сейчас изменения сохраняются только в памяти браузера

## Структура файлов

```
admin.html              - Главная страница админки
scripts/admin-editor.js - Логика админки
styles/admin-editor.css - Стили админки
data/projects.json      - Данные проектов
```

## Примечания

- Админка визуально повторяет главную страницу сайта
- Все изменения видны сразу в режиме редактирования
- После сохранения сайт автоматически обновляется
