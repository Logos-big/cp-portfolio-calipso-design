# Быстрое обновление из Figma

## Автоматическое обновление

После того как вы обновили макет в Figma, просто запустите:

### Вариант 1: Двойной клик на файл
```
обновить-сайт.bat
```

### Вариант 2: Через PowerShell
```powershell
.\scripts\auto-update.ps1
```

## Что делает скрипт:

1. ✅ Получает свежие данные из Figma API
2. ✅ Генерирует HTML строго по макету
3. ✅ Обновляет CSS переменные
4. ✅ Автоматически пушит изменения в GitHub
5. ✅ Сайт обновляется на GitHub Pages через 1-2 минуты

## Если возникает ошибка 429 (Rate limit):

- Подождите 1-2 минуты и попробуйте снова
- Скрипт автоматически использует кэшированные данные, если API недоступен

## Ручное обновление:

Если автоматический скрипт не работает:

```powershell
# 1. Синхронизация данных из Figma
.\scripts\figma-sync.ps1

# 2. Генерация HTML
.\scripts\generate-html-from-figma.ps1

# 3. Обновление на GitHub
git add .
git commit -m "Update from Figma"
git push origin main
```
