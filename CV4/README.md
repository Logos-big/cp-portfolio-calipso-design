# Calipso Design Portfolio

Автоматическая генерация HTML из макета Figma.

## Быстрый старт

1. Настройте `scripts/figma-config.json`:
   - `fileKey` - ключ файла Figma
   - `token` - токен Figma API
   - `nodeIds` - ID узла "Main" (обычно `["67:5539"]`)

2. Запустите обновление:
   ```
   update.bat
   ```

## Как это работает

Скрипт **строго следует макету Figma**:
- ✅ Генерирует только элементы, которые есть в макете
- ✅ Применяет только стили из макета
- ✅ Не добавляет ничего лишнего

## Структура проекта

```
.
├── index.html              # Генерируется автоматически
├── styles/
│   ├── figma-variables.css # CSS переменные из Figma
│   └── main.css           # Дополнительные стили
├── scripts/
│   ├── update.ps1         # Основной скрипт обновления
│   └── figma-config.json  # Конфигурация Figma
└── update.bat             # Запуск обновления
```

## Обновление из Figma

После изменения макета в Figma просто запустите:
```
update.bat
```

Скрипт автоматически:
1. Получит данные из Figma
2. Сгенерирует CSS переменные
3. Создаст HTML строго по макету
4. Отправит изменения в GitHub (если настроен Git)

## Важно

- Скрипт работает **только с макетом "Main"** из Figma
- Все стили берутся **напрямую из Figma** - никаких модификаций
- Если элемент не в макете - он не появится в HTML
