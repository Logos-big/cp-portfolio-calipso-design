# Интеграция Figma с Cursor

## Способ 1: Экспорт ресурсов из Figma (самый простой)

### Шаг 1: Экспорт изображений и иконок
1. Откройте ваш дизайн в Figma
2. Выберите элемент (изображение, иконка)
3. Правый клик → **Export** или используйте панель справа
4. Выберите формат (PNG, SVG, JPG)
5. Сохраните в папку `images/` вашего проекта

### Шаг 2: Экспорт CSS стилей
1. Выберите элемент в Figma
2. В правой панели найдите раздел **Code**
3. Скопируйте CSS свойства
4. Вставьте в ваш `styles/main.css`

### Шаг 3: Получение цветов и шрифтов
1. Выберите элемент
2. В правой панели скопируйте:
   - **Цвета** (HEX код)
   - **Шрифты** (название и размер)
3. Добавьте в CSS переменные:

```css
:root {
    --primary-color: #ваш-цвет;
    --font-family: 'Ваш шрифт', sans-serif;
}
```

## Способ 2: Figma Dev Mode (для подписчиков)

Если у вас есть подписка Figma:
1. Откройте файл в Figma
2. Нажмите **Dev Mode** (вверху справа)
3. Вы увидите CSS, размеры, отступы для каждого элемента
4. Скопируйте код и используйте в проекте

## Способ 3: Плагины Figma для экспорта кода

### Плагины для установки в Figma:
1. **Figma to Code** - экспорт в HTML/CSS
2. **Figma to React** - для React компонентов
3. **Figma Export** - массовый экспорт ресурсов

### Как установить плагин:
1. В Figma: меню → **Plugins** → **Browse plugins**
2. Найдите нужный плагин
3. Нажмите **Install**
4. Запустите через меню **Plugins** → **Ваш плагин**

## Способ 4: Figma API (для автоматизации)

Если нужно автоматизировать процесс:

1. Получите Figma Access Token:
   - Figma → Settings → Account → Personal Access Tokens
   - Создайте новый токен

2. Используйте Figma API для получения данных:
   ```javascript
   // Пример получения цветов из Figma
   fetch('https://api.figma.com/v1/files/FILE_KEY', {
     headers: {
       'X-Figma-Token': 'YOUR_TOKEN'
     }
   })
   ```

## Способ 5: Ручная структура проекта

Создайте структуру папок для ресурсов из Figma:

```
project/
├── images/
│   ├── icons/          # Иконки из Figma
│   ├── illustrations/ # Иллюстрации
│   └── photos/         # Фотографии
├── fonts/              # Шрифты (если кастомные)
└── styles/
    └── main.css        # Стили из Figma
```

## Рекомендуемый рабочий процесс:

1. **Дизайн в Figma** → Создайте/отредактируйте дизайн
2. **Экспорт ресурсов** → Сохраните изображения в `images/`
3. **Копирование стилей** → Скопируйте CSS из Figma Dev Mode
4. **Реализация в Cursor** → Вставьте код в проект
5. **Тестирование** → Проверьте на GitHub Pages

## Полезные инструменты:

- **Figma Desktop App** - для работы с файлами
- **Figma Browser Extension** - для быстрого доступа
- **Figma Community** - готовые компоненты и шаблоны
