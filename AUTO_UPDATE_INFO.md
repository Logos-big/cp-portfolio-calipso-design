# Автоматическое обновление из Figma

## ✅ Настроено автоматическое обновление!

Теперь скрипт `scripts/auto-update.ps1` автоматически:
1. ✅ Получает свежие данные из Figma
2. ✅ Обновляет HTML страницу
3. ✅ **Автоматически обновляет сайт на GitHub Pages**

## Как использовать:

### Вариант 1: Запустить bat-файл (самый простой)

**Дважды кликните на файл:** `update-from-figma.bat`

### Вариант 2: Запустить PowerShell скрипт

В терминале выполните:
```powershell
.\scripts\auto-update.ps1
```

## Что происходит автоматически:

1. **Синхронизация из Figma**
   - Подключение к Figma API
   - Получение данных макета "Main"
   - Сохранение в `figma-data.json`

2. **Генерация HTML**
   - Поиск узла "Main" в данных
   - Генерация HTML структуры
   - Обновление `index.html`

3. **Автоматическое обновление сайта** ⭐
   - Добавление изменений в Git
   - Создание коммита
   - Отправка на GitHub
   - Сайт обновится через 1-2 минуты!

## Результат:

После выполнения скрипта:
- ✅ Локальные файлы обновлены
- ✅ Изменения отправлены на GitHub
- ✅ Сайт автоматически обновится на GitHub Pages

**Ваш сайт:** https://Logos-big.github.io/cp-portfolio-calipso-design/

---

## Если нужно обновить только из Figma (без GitHub):

Используйте старый скрипт:
```powershell
.\scripts\figma-sync.ps1
```

Этот скрипт только синхронизирует данные, но не обновляет сайт.
